# Сетка — конструктор кроссвордов

Один HTML-файл, в котором можно собрать кроссворд из своих вопросов и отправить его кому угодно.

## Что умеет

- **Конструктор.** Вводите пары «ответ + подсказка» — сетка строится автоматически, переплетая слова по общим буквам.
- **Интерактивная игра.** Клик по клетке, ввод с клавиатуры, проверка ответов, подсказки, прогресс-бар, экран победы.
- **Поделиться.** Кнопка «Поделиться» генерирует ссылку и короткий код. Получатель открывает ссылку и видит только кроссворд — без доступа к конструктору и ответам.
- **Печать.** Кнопка «Печать» выводит пустую сетку с подсказками — удобно для раздачи на тренинге или уроке.

## Быстрый старт

1. Скачайте `index.html` (или склонируйте репозиторий).
2. Откройте файл в браузере.
3. Впишите свои слова и подсказки, нажмите **Собрать кроссворд**.
4. Нажмите **Поделиться** → скопируйте ссылку или код.

Для публикации в интернете выложите `index.html` на GitHub Pages, Netlify Drop или любой статический хостинг.

## Технические детали

- Один файл, ноль зависимостей, работает офлайн.
- Кроссворд целиком закодирован в URL — сервер и база данных не нужны.
- Русский и латинский алфавит.
- Адаптивная вёрстка: десктоп, планшет, телефон.
